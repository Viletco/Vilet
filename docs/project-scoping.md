# Project scoping and lead summary

The project tool supports Website, Website redesign, AI/business automation, Custom software, Ongoing support, and Not sure. It asks for a category, organization type, problem, outcome, and only a few optional contextual fields. It never requests credentials, payment data, health information, IDs, or private customer records.

Service recommendations use approved service names. Output is labeled **Preliminary project brief — requires Vilét review** and contains no price, binding scope, legal terms, acceptance, or guaranteed schedule. The growth advisor similarly recommends a simple website/process improvement when AI is not justified.

Copy is local. Selecting **Review in Contact** explicitly places only the visible summary and a generic goal in `sessionStorage`, navigates to Contact, populates those two fields, and deletes the stored handoff. The visitor reviews, edits, completes identity fields, and submits through the existing Contact backend. Nothing submits automatically; disabled contact delivery remains honest.
